## 概述

Gemini CLI 是 Google 推出的开源 AI 命令行助手，基于 Gemini 模型，提供强大的终端 AI 交互体验。

## 核心能力

- **对话式编程**：在终端与 Gemini 进行自然语言交互
- **代码理解**：深度分析项目结构和代码逻辑
- **多模态支持**：支持图片、文件等多种输入形式
- **扩展生态**：通过 MCP (Model Context Protocol) 连接外部工具
- **自定义 Agent**：创建专用的 AI Agent 处理特定任务

## 快速入门

```bash
# 安装 Gemini CLI
npm install -g @google/gemini-cli

# 在项目目录启动
cd your-project
gemini
```

## 高级技巧

- 使用 MCP 服务器扩展工具能力
- 通过 `.gemini/` 目录配置项目级设定
- 利用 System Instructions 定制行为模式
